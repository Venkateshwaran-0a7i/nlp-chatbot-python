# NLP-Based Chatbot 🤖

An intelligent chatbot built using Python, NLP techniques, TF-IDF vectorization, and cosine similarity.

## Features
- Intent-based conversation
- NLP preprocessing with NLTK
- Confidence-based fallback responses
- Clean project structure

## Tech Stack
- Python
- NLTK
- Scikit-learn

## How to Run
```bash
pip install -r requirements.txt
python main.py
